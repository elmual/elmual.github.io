Home site (HTML)

The history of the development of the HTML language began in 1989 with the proposal of a hypertext document system by Tim Berners-Lee, a student at Oxford University. In 1990, he called it the World Wide Web. One of the parts that made up the system was to insert a hypertext tab. It was founded in 1990 when Berners-Lee created a browser that allowed viewing hypertext documents. Finally, in 1995, the first version of the language - HTML 1.0 - was released. Only after HTML 2.0 was completed in 1995, did the HTML language become standardized. At that time, most browsers supported the new version of HTML. In 1996, all browsers supported the HTML 3.2 version of the language (HTML 3.0 introduced tables). As a result, web design has risen to a higher level. New opportunities have emerged, ushering in the era of web design. The next version of the language, HTML 4.01, was standardized in 1999. After that, the development of the language stopped. Because, in fact, nothing else could be done. New technologies have emerged that further satisfy modern internet users.
